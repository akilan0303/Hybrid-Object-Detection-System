Hybrid object detection system using YOLOv8 and Faster R-CNN for images, videos, and real-time webcam.

Features

- Detects objects using YOLOv8 and Faster R-CNN
- Hybrid predictions combining both models
- Shows object count in each frame
- Works with images, videos, and webcam

Technologies Used

- Python
- PyTorch
- TorchVision
- Ultralytics YOLOv8
- OpenCV
- NumPy
- Matplotlib

How to Run

1. Clone the repository
2. Open Hybrid_Object_Detection_System.ipynb in Colab or Jupyter
3. Run all cells
4. Upload an image/video or use webcam to test
